# FastAPI Async Blog

A FastAPI asynchronous blog project that leverages the power of FastAPI, SQLAlchemy, PostgreSQL, and Alembic to provide a modern and efficient blog application.

## Requirements

- Python 3.11
- FastAPI (latest version)
- SQLAlchemy
- PostgreSQL
- Alembic
- asyncpg
